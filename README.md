# Прогресс-бар

Интерактивный компонент прогресс-бар, реализованный на JavaScript, HTML, CSS с возможностью управления состоянием через API.
Проект адаптирован под разные ориентации экрана.

## Возможности проекта

- Отображение прогресса в процентах.
- Управление через пользовательский ввод.
- Управление через API:
    - getValue() - получить текущее значение;
    - setValue() - задать новое значение;
    - setAnimated() - включить/выключить вращение;
    - setHidden() - скрыть или показать прогресс-бар;
